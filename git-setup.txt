# Git Setup Commands (Copy-Paste These)

# 1. Initialize git in your project folder
git init

# 2. Add all files
git add .

# 3. Make first commit
git commit -m "Complete AI Snake Game with Kiro specs - Retro Revival Challenge"

# 4. Create repository on GitHub first, then add remote
# Replace 'yourusername' with your actual GitHub username
git remote add origin https://github.com/yourusername/ai-snake-game-retro-revival.git

# 5. Push to GitHub
git push -u origin main

# If you get an error about 'main' branch, try:
git branch -M main
git push -u origin main

# 6. To update later (after making changes):
git add .
git commit -m "Update documentation and blog content"
git push
